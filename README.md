## 📌 Sobre o Projeto

O **FinanceHUB** é um sistema web desenvolvido para a organização e gerenciamento de documentos financeiros, como notas fiscais, recibos e comprovantes.

A solução foi projetada para atender pequenas empresas e profissionais autônomos que enfrentam dificuldades na organização financeira, oferecendo uma plataforma simples, acessível e eficiente.

Este projeto foi desenvolvido no contexto de um **Projeto de Extensão Universitária**, com foco na aplicação prática do conhecimento acadêmico em cenários reais.

---

## 🎯 Objetivo

O FinanceHUB tem como objetivo principal:

- Centralizar documentos financeiros em um único ambiente  
- Reduzir erros operacionais  
- Melhorar o controle de processos financeiros  
- Facilitar o acesso às informações  

---

## 🚀 Funcionalidades

- 📂 Upload de documentos financeiros  
- 🔎 Busca e filtragem de arquivos  
- 🗂️ Organização por categorias  
- 👤 Gerenciamento de usuários  
- 🔐 Controle de acesso  
- 📊 Visualização estruturada das informações

## 🏢 Aplicação Prática

O sistema foi desenvolvido com foco em aplicação real, podendo ser utilizado por:

- Microempresas  
- Profissionais autônomos  
- Negócios informais  
- Setores administrativos  

Como estudo de caso, a solução foi direcionada à empresa:

**FROST CREATIVE LTDA**

Empresa do setor de economia criativa, com atuação em:

- Produção audiovisual  
- Marketing digital  
- Desenvolvimento de soluções tecnológicas  

A aplicação do sistema contribui para a organização de documentos financeiros e melhoria dos processos administrativos.

---

## 🌍 Impacto Social

O FinanceHUB contribui diretamente para a melhoria da gestão financeira de pequenos negócios, especialmente aqueles que não possuem acesso a sistemas especializados.

Entre os principais impactos, destacam-se:

- Inclusão digital  
- Profissionalização da gestão  
- Redução de erros operacionais  
- Acesso simplificado à tecnologia  

Dessa forma, o projeto cumpre o papel da extensão universitária ao gerar benefícios concretos para a sociedade.
